PROBLEM: "Camera not working"
SOLUTION: 
  1. Install: pip install opencv-python
  2. Check if webcam is connected
  3. Make sure no other app is using camera

PROBLEM: "Voice not working"
SOLUTION:
  1. Install: pip install pyttsx3 SpeechRecognition pyaudio
  2. Check microphone in Windows Settings
  3. Test microphone volume

PROBLEM: "GUI won't start"
SOLUTION:
  1. Check Python is installed: python --version
  2. Install tkinter: pip install tkinter
  3. Make sure you're in SARA folder

PROBLEM: "API key error"
SOLUTION:
  1. Set key: $env:GROQ_API_KEY="your_key"
  2. Or just use fallback mode (works great!)
  3. Get free key: https://console.groq.com

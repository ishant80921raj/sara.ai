================================================================================
                          TROUBLESHOOTING GUIDE
================================================================================

ISSUE: Can't hear SARA responses
  SOLUTION 1: Check speakers are enabled and volume is up
  SOLUTION 2: Run test: python test_hinglish.py
  SOLUTION 3: Verify pyttsx3 is working: python voice_output.py

ISSUE: SARA won't recognize voice input  
  SOLUTION 1: Speak clearly and loud enough
  SOLUTION 2: Check microphone is connected and working
  SOLUTION 3: Reduce background noise
  SOLUTION 4: Check internet (for Google Speech API)

ISSUE: Hinglish not being recognized
  SOLUTION 1: Verify speech recognition is working
  SOLUTION 2: Ensure microphone volume is high
  SOLUTION 3: Speak complete command clearly
  SOLUTION 4: Use supported keywords from reference guide

ISSUE: SARA speaking too fast or slow
  SOLUTION 1: Edit config.py
  SOLUTION 2: Change SPEECH_RATE value (150 = normal)
  SOLUTION 3: Restart SARA

ISSUE: Can't start SARA in voice mode
  SOLUTION 1: Check Python is installed (python --version)
  SOLUTION 2: Check dependencies: pip install -r requirements.txt
  SOLUTION 3: Check microphone is connected
  SOLUTION 4: Run in interactive mode first: python main.py --mode interactive
